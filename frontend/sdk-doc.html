📦 SIMBIOSIS SDK

SIMBIOSIS permite mostrar anuncios geolocalizados en videojuegos de forma automática. Este SDK es una forma rápida de integrarlos en cualquier juego HTML5.

🚀 Integración rápida

<div id="geo-ad-slot"></div>
<script src="https://geoadvertising-frontend.onrender.com/sdk.js"></script>
<script>
  GeoAd.getAd("Madrid", document.getElementById("geo-ad-slot"));
</script>
