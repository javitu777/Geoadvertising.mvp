📦 GeoAdvertising SDK

Integrate real-time geolocated ads directly into your HTML5 games with just a few lines of code. This SDK allows you to fetch ads by city and display them dynamically in your game interface.

🚀 Getting Started

Add the following code to your HTML game page:

<div id="geo-ad-slot" style="position:absolute; bottom:10px; left:10px;"></div>

<script src="https://geoadvertising-frontend.onrender.com/sdk.js"></script>
<script>
  GeoAd.getAd("Madrid", document.getElementById("geo-ad-slot"));
</script>

🌍 Targeting

Ads are served based on the city you provide in GeoAd.getAd("CITY"). Future versions will support advanced targeting by map zones.

🔧 SDK Function Reference

  • GeoAd.getAd(location, containerElement) – Loads and renders an ad for the given location inside the provided DOM container.
